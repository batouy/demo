es6加载模式：请查看控制台
1. 可以直接通过`script`标签来加载外部脚本，需要加上`type="module" `，这个属性会自动延迟异步加载
2. 也可以通过`script`标签内嵌在网页中,同样需要加上` type="module"`，才能起作用
3. 最常用的还是通过外链加载
4. 如果使用`node`加载，使用 

node --experimental-modules index.mjs
其中后缀为.mjs的文件,是为了兼容node环境执行。如果只是浏览器环境执行，可以使用.js后缀

import会提前执行,这里会先加载math,然后输出`loader-index.mjs` 

        console.log('loader-index.mjs')
        import {
          math
        } from './math.js'

import命令加载其他模块(.js后缀不可以省略，需要提供绝对URL或相对URL)
export命令输出对外接口
加载多次，也只执行一次。
